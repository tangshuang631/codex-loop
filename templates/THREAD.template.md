# Primary Thread Contract

## Goal

Keep one Codex desktop thread as the primary visible history for this loop.

## Required metadata

- Thread title
- Thread id
- Current run id
- Last heartbeat summary
- Last verification summary

## Rules

- Keep work in one primary thread whenever possible.
- Use local mirror files for recovery, not as a replacement for the main Codex transcript.
- Record any handoff immediately if a new thread is ever required.
